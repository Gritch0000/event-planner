## Основні сутності

### `Event` (Подія)
| Поле | Тип | Опис |
|------|------|------|
| `id` | `string` / `integer` | Унікальний ідентифікатор події |
| `title` | `string` | Назва події |
| `description` | `string` | Опис події |
| `date` | `string` (ISO 8601) | Дата та час проведення |
| `location` | `string` | Місце проведення |

### `Participant` (Учасник)
| Поле | Тип | Опис |
|------|------|------|
| `id` | `string` / `integer` | Унікальний ідентифікатор учасника |
| `name` | `string` | Повне ім’я учасника |
| `email` | `string` | Електронна пошта |
| `eventId` | `string` / `integer` | Ідентифікатор події, до якої прив’язаний учасник |

---

## Ендпоінти (API)

| Метод | Шлях | Опис | Статус відповіді |
|--------|--------|--------|-------------------|
| `GET` | `/events` | Отримати список усіх подій | `200 OK` |
| `GET` | `/events/:id` | Отримати одну подію за її ID | `200 OK` / `404 Not Found` |
| `POST` | `/events` | Створити нову подію | `201 Created` / `400 Bad Request` |
| `PATCH` | `/events/:id` | Частково оновити дані існуючої події | `200 OK` / `404 Not Found` |
| `DELETE` | `/events/:id` | Видалити подію з системи | `200 OK` / `404 Not Found` |
